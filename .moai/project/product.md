# Resource Monitor PyQT - 제품 문서

## 프로젝트 개요

**Resource Monitor PyQT**는 Python과 PyQt5를 기반으로 구축된 실시간 시스템 자원 모니터링 애플리케이션입니다. CPU, 메모리, GPU, FPS 등 주요 시스템 자원을 1초 간격으로 실시간 측정하고 Qt Quick(QML) 기반의 반응형 UI로 시각화합니다.

MVVM(Model-View-ViewModel) 아키텍처 패턴을 채택하여 UI와 비즈니스 로직을 명확하게 분리하며, GPU 모니터링 모듈의 경우 GPUtil 라이브러리가 없는 환경에서도 자동으로 대체 동작하는 우아한 성능 저하(Graceful Degradation) 전략을 사용합니다.

---

## 대상 사용자

| 사용자 그룹 | 설명 |
|---|---|
| 개발자 | 애플리케이션 개발 중 시스템 자원 사용량을 실시간으로 확인하려는 소프트웨어 엔지니어 |
| 시스템 관리자 | 서버 또는 워크스테이션의 자원 상태를 지속적으로 모니터링해야 하는 운영 담당자 |
| 성능 분석가 | 게임, 렌더링, 머신러닝 워크로드 등 자원 집약적 작업의 성능을 프로파일링하는 엔지니어 |
| PyQt5 학습자 | MVVM 패턴과 QML 연동 방식을 학습하려는 Python GUI 개발 입문자 |

## 핵심 기능

### CPU 모니터링
- `psutil` 라이브러리를 통한 CPU 사용률(%) 실시간 측정
- 멀티코어 환경 지원

### 메모리 모니터링
- 시스템 RAM 사용률(%) 및 사용량 실시간 측정
- `psutil` 기반의 안정적인 데이터 수집

### GPU 모니터링
- `GPUtil` 라이브러리를 활용한 GPU 사용률 측정
- GPU가 없거나 GPUtil 미설치 환경에서는 자동으로 비활성화되는 선택적 모듈

### FPS 추적
- 애플리케이션 프레임 레이트(초당 프레임 수) 실시간 측정
- UI 렌더링 성능 모니터링에 활용 가능

### 실시간 UI 업데이트
- QTimer 기반 1초 주기 자동 갱신
- QML ListView를 활용한 모든 자원 정보의 통합 표시
- PyQt 신호/슬롯(Signal/Slot) 메커니즘을 통한 데이터 바인딩

## 주요 사용 사례

**개발 환경 모니터링**
코드 컴파일, 테스트 실행, 빌드 작업 중 CPU 및 메모리 소비량을 추적하여 병목 지점을 식별합니다.

**게임 및 렌더링 성능 분석**
GPU 사용률과 FPS를 동시에 관찰하여 그래픽 집약적 작업의 성능 한계를 파악합니다.

**머신러닝 워크로드 모니터링**
모델 학습 중 GPU 사용률 및 메모리 점유율을 실시간으로 확인하여 하드웨어 활용 효율을 최적화합니다.

**PyQt5/QML 학습**
MVVM 패턴, QML과 Python 간의 데이터 바인딩, PyQt 신호/슬롯 활용법을 실제 동작하는 코드로 학습합니다.
